HTML - Text Links

Linking Documents

A link is specified using HTML tag <a>. This tag is called anchor tag and anything between the opening <a> tag and the closing </a> tag becomes part of the link and a user can click that part to reach to the linked document. Following is the simple syntax to use <a> tag.

Example

Sample Code

<!DOCTYPE html>
<html>
  <head>
    <title>Hyperlink Example</title>
  </head>
  <body>
    <p>Click the following link</p>
    <a href="www.LearnersPoint.com" target="_self">LearnsPoint</a>
  </body>
</html>

This will produce the following result −

Click the following link

LearnersPoint

Attribute

Target Attribute

We have used target attribute in our previous example. This attribute is used to specify the location where linked document is opened. Following are the possible options −

Sr.No Option Description
1 _blank Opens the linked document in a new window or tab.
2 _self Opens the linked document in the same frame.
3 _parent Opens the linked document in the parent frame.
4 _top Opens the linked document in the full body of the window.
5 targetframe Opens the linked document in a named target frame.
Example

Sample Code

<!DOCTYPE html>
<html>
  <head>
    <title>Hyperlink Example</title>
    <base href="https://www.tutorialspoint.com/" />
  </head>
  <body>
    <p>Click any of the following links</p>
    <a href="/html/index.htm" target="_blank">Opens in New</a> |
    <a href="/html/index.htm" target="_self">Opens in Self</a> |
    <a href="/html/index.htm" target="_parent">Opens in Parent</a> |
    <a href="/html/index.htm" target="_top">Opens in Body</a>
  </body>
</html>

This will produce the following result −

Click any of the following links

Opens in New | Opens in Self | Opens in Parent | Opens in Body

HTML Email Tag

Set Email Link

HTML <a> tag provides you option to specify an email address to send an email. While using <a> tag as an email tag, you will use mailto: email address along with href attribute. Following is the syntax of using mailto instead of using http.

Example

Sample Code

<a href = "mailto: abc@example.com">Send Email

This will produce the following result −

Send Email

HTML - Image Links

Set Image Link

It's simple to use an image as hyperlink. We just need to use an image inside hyperlink at the place of text as shown below −

Example

Sample Code

<!DOCTYPE html>
<html>
  <head>
    <title>Image Hyperlink Example</title>
  </head>
  <body>
    <p>Click the following link</p>
    <a href="https://www.LearnsPoint.com" target="_self">     <img src="/images/logo.png" alt="Tutorials Point" border="0"/>     </a>
  </body>

Download Links

Download Doc

You can create text link to make your PDF, or DOC or ZIP files downloadable. This is very simple; you just need to give complete URL of the downloadable file as follows −

Example

Sample Code

<!DOCTYPE html>
<html>
  <head>
    <title>Hyperlink Example</title>
  </head>
  <body>
    <a href="https://www.LearnersPoint.com/page.pdf">Download PDF File</a>
  </body>
</html>

This will produce the following result −


Download PDF File

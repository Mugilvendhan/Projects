Quiz Level - 1

Question 1: What does HTML stand for?

a) Hyper Text Markup Language
b) High Tech Modern Language
c) Hyperlink and Text Markup Language

Question 2: Which HTML tag is used to create a hyperlink?

a) <link>
b) <a>
c) <hyperlink>

Question 3: What is the correct HTML element for inserting a line break?

a) <break>
b) <br>
c) <line>

Question 4: Which HTML tag is used to define an unordered list?

a) <ul>
b) <ol>
c) <list>

Question 5: What does CSS stand for?

a) Creative Style Sheets
b) Cascading Style Sheets
c) Computer Style Sheets

Question 6: In HTML, which tag is used to define the structure of an HTML document?

a) <header>
b) <body>
c) <html>

Question 7: What is the correct HTML tag for creating an ordered list?

a) <ol>
b) <ul>
c) <li>

Question 8: Which HTML element is used to define the title of an HTML document?

a) <head>
b) <title>
c) <h1>

Question 9: What does the "href" attribute in an <a> tag specify?

a) The heading of the link
b) The location of the linked document
c) The style of the link

Question 10: Which HTML tag is used to display a horizontal line?

a) <line>
b) <hr>
c) <hl>

Question 11: What is the purpose of the <meta> tag in HTML?

a) To create hyperlinks
b) To define metadata about an HTML document
c) To add images to a webpage

Question 12: Which HTML tag is used to define a table row?

a) <tr>
b) <td>
c) <th>

Question 13: In HTML, which tag is used to create a numbered list?

a) <ul>
b) <li>
c) <ol>

Question 14: What is the purpose of the <iframe> tag in HTML?

a) To create a new webpage
b) To embed another webpage within the current webpage
c) To add images to a webpage

Question 15: Which HTML tag is used to create a hyperlink to send an email?

a) <a>
b) <mailto>
c) <email>

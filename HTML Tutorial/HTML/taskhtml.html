Task Level - 1

HTML Tasks

  1. Basic Web Page:

    Create a basic HTML web page with a header, paragraph, and a list.

  2. HTML Forms:

    Build a contact form with fields for name, email, and a message. Include a submit button.

  3. Image Gallery:

    Create a simple image gallery using HTML and display multiple images in a grid.

  4. Hyperlinking:

    Build a webpage with various types of hyperlinks, including internal links and links to external websites.

  5. Table Creation:

    Create an HTML table to display information such as student grades, products, or any other dataset.

  6. Ordered and Unordered Lists:

    Create both ordered and unordered lists to organize information on a webpage.

  7. Embed Multimedia:

    Embed a YouTube video or audio file (e.g., an MP3) on an HTML page.

  8. HTML5 Semantic Elements:

    Develop a webpage using HTML5 semantic elements like <header>, <nav>, <section>, <article>, and <footer>.

  9. HTML Forms and CSS Styling:

    Create a form with various input types and apply CSS styles for layout and design.

  10. Responsive Web Design:

    Build a simple webpage and make it responsive using media queries to adapt to different screen sizes.
